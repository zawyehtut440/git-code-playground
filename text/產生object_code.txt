當遇到label field有symbol的時候, 找找undefined_symtab中有沒有此symbol
有在undefined_symtab的話, 依據undefined_symtab[label]的內容寫出T record
沒有在undefined_symtab的話, 把此label加入symtab(要注意加入symtab時, 可能會發生的錯誤)
接著把mnemonic跟operand翻譯成object code

如果mnemonic是optab中的指令, 翻譯成opcode
    如果有operand
        查symtab, 如果在symtab
            把operand位址加上object code
        查symtab, 如果不在symtab
            把key: operand, value: locctr的位址+1存到undefined_symtab中
            把operand的部分補四個0, 加到object code
    如果沒有operand
        把operand的部分補四個0, 加到object code
    locctr += 3
如果mnemonic是BYTE
    如果是C字串
        把字串內的內容, 翻譯成ASCII code
        把ASCII加到object code
        locctr += length of string content
    如果是X字串
        把字串內的內容, 加入到object code
        locctr += length of string content / 2
如果mnemonic是WORD
    把operand轉成16進位, 6位數字, 不夠6位前面補0, 然後加到object code
    locctr += 3
如果mnemonic是RESB
    locctr += operand
如果mnemonic是RESW
    locctr += 3 * operand

寫T record
如果mnemonic是RESB或RESW
    如果T record有object code(len(t_record)>1)
        算出這個t_record的長度, 然後加到T_RECORD
        初始化新的t_record
如果mnemonic是BYTE
    計算object code的長度, 如果t_record的長度加上object code的長度>60
        object code兩個兩個算加到t_record中, t_record不能超過60(可以等於)
        然後寫新的t_record, t_record = ["T", "上一個t_record start address + len(object code)"]
    否則
        直接把object code加到t_record
其他(WORD, OPCODE指令)
    if object code will not fit into the current Text record then
        begin
            write Text record to object program
            initialize new Text record
        end
    add object code to Text record
